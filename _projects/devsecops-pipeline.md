## Overview

Designed and implemented a DevSecOps pipeline to integrate automated security testing directly into the software development lifecycle (SDLC).

## Components

*   **SAST (Static Application Security Testing):** Integrated tools to scan source code for vulnerabilities during the build phase.
*   **DAST (Dynamic Application Security Testing):** Configured automated dynamic scans against staging environments.
*   **Container Security:** Automated vulnerability scanning of Docker images before pushing to the registry.

## Impact

*   Reduced the number of vulnerabilities reaching production by shifting security left.
*   Improved developer awareness by providing immediate feedback on insecure coding practices.
