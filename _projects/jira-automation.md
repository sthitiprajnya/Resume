## Overview

Developed a suite of automation tools using Python 3.10 and the JIRA REST API to streamline vulnerability management workflows.

## Features

*   **Automated Ticket Creation:** Parses vulnerability scanner reports (Nessus, Burp Suite) and automatically creates formatted JIRA tickets with relevant context and remediation steps.
*   **SLA Tracking:** Monitors ticket status against defined SLAs and triggers automated alerts (via email/Slack) for approaching breaches.
*   **Dashboard Generation:** Aggregates risk data to build automated weekly dashboards for management visibility.

## Results

*   Improved high-risk finding closure rates by 35%.
*   Reduced manual reporting effort for the security team by ~8 hours per week.
